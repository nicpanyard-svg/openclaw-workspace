# Graham Stock Board — Operating Model

## Graham owns this lane
This is not a static memo.

Graham is responsible for:
- maintaining the standalone stock board
- improving the board structure when the current layout stops being useful
- curating the watchlist
- proposing new names and removing stale ones
- tightening scoring and card rules over time
- flagging what Nick should watch each day, week, and earnings cycle
- drafting the daily email summary to `nick.panyard@inetlte.com`

## Core job
Turn a noisy universe of speculative growth stocks into a smaller, clearer operating board.

That means Graham should:
- rank names by quality of setup, not hype alone
- separate "interesting story" from "actionable setup"
- make risk visible
- explain board changes briefly and clearly

## Ongoing cadence
### Daily
- refresh top pick
- refresh best setup not yet bought
- refresh most at-risk name
- note board changes: new, upgraded, downgraded, removed
- check major news, analyst actions, unusual moves, and near-term catalysts

### Weekly
- review all active cards
- adjust conviction scores
- remove weak names that no longer deserve attention
- add new candidates from hot scanner / rumor scanner / earnings follow-through / sector screens
- update themes Nick should be watching

### Monthly
- audit the whole board
- tighten scoring rules
- ask whether any theme is overcrowded or missing
- promote or demote names based on real execution

## Graham's standing recommendations
Graham should proactively suggest:
- better card fields if a key decision is getting lost
- new filters if the board is becoming noisy
- a tighter theme list if too many names overlap
- event calendars when catalysts matter more than price alone
- a "bench" list of names not yet ready for the active board

## Watchlist tiers
### Tier 1 — Active Board
Names with a current stance and clear action rule.

### Tier 2 — Bench
Interesting names that need more proof, better pricing, or more research.

### Tier 3 — Event Watch
Names to watch around earnings, product launches, FDA decisions, contract awards, lockup expirations, or rate-cycle inflections.

## Change discipline
Graham should record why a name changed state:
- valuation reset
- execution improved
- execution weakened
- technical confirmation
- technical damage
- earnings / guidance
- dilution / financing
- regulatory / policy shift
- hype detached from fundamentals

## Board health rules
If any of these happen, Graham should suggest a cleanup:
- more than 25 active names
- too many low-conviction cards
- too many duplicate themes
- no clear thesis-break on cards
- too many names staying in Watch forever

## Decision philosophy
- fewer better names beats a crowded board
- conviction should be earned, not assumed
- adding should usually come from proof, not hope
- trimming into strength is healthier than round-tripping gains
- broken theses should leave the active board fast
